🧮 NYG Captcha 

A simple Captcha built with pure HTML, CSS and JavaScript.

📸 Preview

<img src="captcha.jpeg"></img>

✨ Features

- 🔐 CAPTCHA generation — creates randomized verification challenges
- 🔄 Reload option — generate a new CAPTCHA instantly
- 🔍 Input validation — checks if the entered value matches the CAPTCHA
- ❌ Error feedback — displays "Invalid CAPTCHA" on incorrect input
- ⚠️ Empty input handling — prompts user when no value is entered
- 🧠 Noise & distortion — adds visual complexity to prevent bots
- ⏱️ Attempt handling — limits or tracks user attempts (optional)
- 📱 Responsive design — works across different screen sizes
